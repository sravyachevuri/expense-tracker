# Expense Tracker Web Application

## Project Description
Expense Tracker is a Java web application developed using Servlets, HTML, and CSS.  
This application helps users add and manage daily expenses along with category-wise expense tracking.

---

## Features
- Add new expenses
- Select expense category
- View all expenses
- Category-wise total calculation
- Total expense calculation
- Input validation for amount
- Simple and responsive UI

---

## Technologies Used
- Java
- Servlets
- HTML
- CSS
- Collections (ArrayList, HashMap)
- Apache Tomcat
- Eclipse IDE

---

## Project Structure
- `AddExpense.java` → Handles adding expenses
- `ViewExpenses.java` → Displays expenses and totals
- `Expense.java` → Expense model class
- `ExpenseData.java` → Stores expense list
- `addExpense.html` → Expense form
- `style.css` → Styling
